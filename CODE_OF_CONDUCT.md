# Code of Conduct

## Our Pledge

We as members, contributors, and maintainers of **Avila Math** pledge to make participation in our project and community a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Technical Conduct

- Write clean, well-documented code
- Test your contributions before submitting
- Follow Rust best practices and idioms
- Respect existing code style and architecture
- Be open to feedback on your contributions

## Enforcement

### Reporting

If you experience or witness unacceptable behavior, please report it by:
- Opening a confidential issue
- Contacting project maintainers directly

### Consequences

Maintainers may take any action they deem appropriate, including:
- Warning the offender
- Temporary ban from the project
- Permanent ban from the project

## Scope

This Code of Conduct applies to:
- All project spaces (GitHub issues, pull requests, discussions)
- Official communication channels
- Public spaces when representing the project
